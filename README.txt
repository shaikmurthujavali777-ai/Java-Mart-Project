Console-Based Java Mini Project

Project Overview
----------------
This is a console-based Java application developed using Object-Oriented Programming (OOP) principles such as
Encapsulation, Inheritance, Polymorphism, and Abstraction.
It simulates a small shopping system where users can register, log in, add items, and manage delivery operations.

Features
--------
- User Authentication – Includes registration and login functionalities.
- Item Management – Allows adding and viewing various product categories like groceries, fruits, vegetables, electronics, etc.
- Delivery Module – Simulates order and delivery functionality.
- OOP Concepts Used:
  - Encapsulation – Data and methods are wrapped within classes.
  - Inheritance – Common item properties inherited by specific item types.
  - Polymorphism – Used for method overriding and dynamic behavior.
  - Abstraction – Abstract classes/interfaces used for item and delivery structure.

Compilation and Execution
--------------------------
Step 1: Compile all Java files
    javac -d . vegetables.java
    javac -d . fruit_items.java
    javac -d . Snacks.java
    javac -d . Softdrinks.java
    javac -d . grocery_items.java
    javac -d . electronics.java
    javac -d . Homeessentials.java
    javac -d . beauty_items.java
    javac -d . Main.java
    javac -d . User.java

Step 2: Run the application
    java User
